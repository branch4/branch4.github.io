Consider Using Cdist

Feb 19th, 2012 | Comments

Today, I’ll write about cdist.

It’s a configuration management system, like puppet or chef or some others.

I think the official website for cdist is  here , and you could get most of the information there. So just read it(:-)

First my cdist version is 2.0.7.

To check the version, type

hogehoge@localhost:~/cdist$ cdist -V

cdist 2.0.7

Now to start with, I need a documentation, and the HTML document is handy.
It’s under ${CDIST}/doc/man/man{1,7} (${CDIST} assumes where you installed cdist)
Open “cdist-quickstart.html” with chrome or something.
