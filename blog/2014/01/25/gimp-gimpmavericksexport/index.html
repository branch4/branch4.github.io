[Gimp] gimp@Mavericksで全可視レイヤを別ファイルにexportする方法

Jan 25th, 2014 | Comments

OS : Mavericks(MacOS X10.9.1)
Gimp version : 2.8.10

このplugin使うと一発。
http://registry.gimp.org/node/28268

1. 上記zipをダウンロード
2. unzipあたりで展開
3. 展開したpythonファイルを下記に移動
  ~/Library/Application Support/GIMP/2.8/plug-ins/
4. 3のファイルに実行権限追加
5. gimpを再起動
6. [File] -> [Export Layers to…]を選択
7. オプションはお好みで

以上でした。
