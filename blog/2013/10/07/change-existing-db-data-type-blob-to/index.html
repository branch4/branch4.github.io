[RoR] Change Existing Db Data Type "Blob" to "Mediumblob"

Oct 7th, 2013 | Comments

Suppose you want to change “thumbnail” column (blob type) to mediumblob.

$ rails g migrate ChangeTableName
$ vi db/migrate/XXXXXXX_change_Table_Name.rb

class ChangeTableName < ActiveRecord::Migration
  def up
    change_column :table_name, :thumbnail, :binary, :limit => 10.megabyte
  end

  def down
    change_column :table_name, :thumbnail, :binary
  end
end

$ rake db:migrate
