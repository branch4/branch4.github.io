--- layout: post title: "[rails] railsサーバを80番ポートで立ち上げちゃう" date: 2014-04-02T18:31:00-07:00 comments: true categories: - Ruby on Rails ---
RVM使ってる人限定。 

      $ rails server
つって起動すると、自動的に3000番であがっちゃいます。 

      $ rails server -p 80
つって起動すると、お前ごときに80番はやらん！
つって怒られます。 

      $ sudo rails server -p 80
つって起動すると、なんだか怒られる。 

      $ rvmsudo rails server -p 80
つって起動すると、rvm関連の環境変数とかもちゃんと読み込んでくれて実行できるようになるらしい。 やってみたらできた。
